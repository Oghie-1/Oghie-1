This is a simple manual on how to clone / push / pull projects from this account

## Open Terminal
  1. "cd" to directory
  >     cd path/to/directory

  2. Initiate a git repository
  >     git init .

  3. Add existing files
  >     git add .

  4. Commit all files (-a) and add a message (-m)
  >     git commit -a -m "commit message goes here"

## Cloning (checking out) someone else's repository
  
  5. Clone the repo with the clone url
>     git clone git@github.com:Oghie-1/{"url"}

## Syncing repository branches with a remote repository
 
 # Make sure you have committed your current state.
 
  6. Get the most up to date code from the remote repo
  >     git pull origin master

  7. Push your local code to the remote repo
  >     git push origin master
